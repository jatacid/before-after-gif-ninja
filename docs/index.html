Installation & Usage

How to Install

  1. Click the "Download Chrome Extension" button above to visit the releases page.
  2. Download the latest .zip file from the releases.
  3. Unzip the downloaded file to a folder on your computer.
  4. Open Chrome and navigate to chrome://extensions/.
  5. Enable "Developer mode" in the top right corner.
  6. Click "Load unpacked" and select the unzipped folder.
  7. Pin the extension icon for quick access.

How to Use

  1. Navigate to any webpage you want to create a before/after GIF from.
  2. Click the extension icon in your browser toolbar to open the side panel.
  3. Click "Select Image A" and drag a rectangle to capture the first area.
  4. Click "Select Image B" and capture the second area for comparison.
  5. Click "Build .gif" to generate the animated GIF with smooth transitions.
  6. Copy the GIF to your clipboard or download it directly.
